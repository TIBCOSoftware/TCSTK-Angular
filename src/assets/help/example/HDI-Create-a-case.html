You can create a new Information Request for the Partner, and the Partner can create Notifications for the Application Owner.

To create a case:

  1. Click +Create Case.
  • select the Partner you want to create the case for. (Application Name)
  • the Partner Application distinguish two Roles Internal (Application Owner) and External (Partner Side)

Partner Internal Case Creators:

  1. create Information Request, multiple Request Types possible

Partner External Case Creators:

  1. new Product Notification
  2. Product recall Notification
